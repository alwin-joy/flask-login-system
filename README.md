# Flask Login System

## Features
- User Registration
- Password Hashing
- Secure Login
- Session Handling
- Logout Functionality
- SQLite Database

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS

## How To Run

1. Install requirements

pip install -r requirements.txt

2. Run Flask app

python app.py

3. Open browser

http://127.0.0.1:5000
